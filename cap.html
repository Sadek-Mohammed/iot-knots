This is the website for displaying the Capstone group's project. It contains all the file deliverables of the group, along with the climate change app. The file deliverables are a portfolio research paper of four distinct chapters, a full-size poster, and an A4 poster. The climate change app uses a self-coded firebase real-time non-relational database to store data of four sensors (air quality sensor, water level sensor, temperature sensor, and a pH sensor) sent by an ESP8266 module. Data are sent via HTTP requests from the ESP and handled by the Firebase Database server. The website then fetches that data from the Firebase Database and puts them into graphs with help from the JavaScript chart.js library. The Simple Statistics JavaScript library calculates mean, median, min, max, standard-deviation, and sample skewness after each new sensor reading and plots them on the website.
